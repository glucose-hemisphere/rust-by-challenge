# Learn Rust

I wanted to learn the Rust programming language, so I decided to ask an AI chatbot to teach it to me through a series of **ten coding challenges**. These coding challenges start small and simple and gradually become bigger and more advanced, culminating in a final project to build a CLI tool.

This repo contains the challenges that I have completed so far, including the description and source code for each challenge.

If you'd like to use this repo to learn Rust for yourself, I'd suggest that you do the following for each challenge:

1. give the challenge description to a chatbot
2. write and run some Rust code to complete the challenge, asking the chatbot for help if you get stuck
3. give your code to the chatbot for verification and coaching

## Challenges (so far)

1. hello_rust
2. guessing_game
3. ownership_puzzle + lifetimes_puzzle
4. model_a_thing
