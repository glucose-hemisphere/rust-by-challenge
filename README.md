# Rust by Challenge

I wanted to learn the Rust programming language, but in such a way that I can start writing code almost immediately, and avoid losing motivation. So I decided to ask an AI chatbot to teach it to me through a series of coding challenges. These coding challenges start small and simple, and gradually become bigger and more advanced, culminating in a final project to build a CLI tool.

This repo contains the challenges that I have completed so far, including the description and source code for each challenge.

This is somewhat inspired by [Rust by Example](https://doc.rust-lang.org/rust-by-example/), but instead of just reading code examples, it involves actually writing and running my own Rust code from the start! This is not intended to be as thorough as [Rust by Example](https://doc.rust-lang.org/rust-by-example/) though, it's just supposed to be enough for me to get to grips with the foundational concepts of the language (as someone who already has some experience with other languages).

If you'd like to use this repo to learn Rust for yourself, I'd suggest that you do the following for each challenge:

1. Read the challenge description.
2. Try to manually write and run some Rust code to complete the challenge.
3. Give the challenge description and your code to a chatbot, then:
    - If you were able to complete the challenge, ask it for verification and review/coaching.
    - If you got stuck, ask it to help you.
4. Once you have completed the challenge successfully, compare your code with my code, and discuss any differences with the chatbot.
